Principais Formatações

Negrito/Destaque

neste caso temos um termo em negrito usadndo a tag<b> (não semantico)

Neste caso temos um termo em destaque utilizando a tag <strong> (semantico)

Italico/Enfase

Neste caso temos um termo em italico utilizando a tag <i> (não semantico)

Neste caso temos um termo em enfase utilizando a tag <em> (semantico)

Texto Marcado

Neste caso temos um texto marcado usando a tag <mark>

Tesxto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno

Texto Deletado

podemos marcar um texto como deletado, para que ele possa ser lido, mas não considerado, utilizando a tag <del>

Texto Inserido

podemos marcar um texto como inserido utilizando a tag <ins> para que possamos enfase e indicar que o texto foi inserido posteriormente

mas tambem podemsos utilzar a tag <u> para inserir (não semantica)

Texto Sobrescrito

podemos marcar um texto como sobre escrito: x20+3. utilizando a tag <sup>

texto subescrito

podemos marcar um texto como sobrescrito: H2O. utilizando a tag <sub>
